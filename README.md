# healthcare-analytics-sql-excel
End-to-end healthcare analytics project including SQL queries, Excel dashboard, and business insights.
Healthcare Analytics Project
Project Overview

This project analyzes hospital admission and treatment data to identify revenue drivers, patient trends, and operational insights.

Tools used:

SQL (SQLite)
DBeaver
Microsoft Excel
Business Questions
Which diagnosis generates the highest revenue?
Which specialty contributes the most revenue?
Which city contributes the highest revenue?
How does revenue trend over time?
Which diagnosis has the longest average length of stay?
Dataset

The project uses four relational tables:

Patients
Doctors
Admissions
Treatments

Relationship:

Patients
   ↓
Admissions
   ↑
Doctors

Admissions
   ↓
Treatments

Key Findings
Revenue Drivers
Cancer generated the highest revenue (1.88M).
Heart Disease ranked second.
Kidney Disease ranked third.
Specialty Analysis
Internal Medicine contributed the highest revenue.
Neurology ranked second.
Geographic Analysis
Depok generated the highest revenue.
Bogor ranked second.
Operational Insight
Kidney Disease had the highest average length of stay.
Admission volume remained relatively stable during 2024–2025.
